# Setup Instructions for RFP-Aggregator

Welcome to the RFP-Aggregator repository! Follow the instructions below to set up the project on your local machine.

## Prerequisites
- Ensure you have installed the following software on your machine:
  - [Node.js](https://nodejs.org/en/download/) (v14 or later)
  - [Git](https://git-scm.com/downloads)

## Cloning the Repository
1. Open your terminal (Command Prompt, PowerShell, or any shell).
2. Run the following command to clone the repository:
   ```bash
   git clone https://github.com/mccreamatthewj/RFP-Aggregator.git
   ```

## Installing Dependencies
1. Navigate into the cloned repository directory:
   ```bash
   cd RFP-Aggregator
   ```
2. Install the required packages using npm:
   ```bash
   npm install
   ```

## Running the Application
- After installing the dependencies, run the application with:
  ```bash
  npm start
  ```

## Additional Information
- If you encounter any issues, please check the [issues section](https://github.com/mccreamatthewj/RFP-Aggregator/issues) for known problems or open a new issue for further assistance.

Happy coding!
