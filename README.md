# RFP Aggregator

## Overview
The RFP Aggregator is a project designed to streamline the process of gathering and analyzing Requests for Proposals (RFPs). This tool aggregates various RFPs from different sources, providing users with a centralized platform to access and manage them effectively.

## Features
- **Aggregation:** Automatically collects RFPs from specified sources.
- **Search and Filter:** Users can search and filter RFPs based on various criteria, including date, source, and type.
- **Notifications:** Users receive notifications for new RFPs in their areas of interest.
- **Analytics Dashboard:** Provides insights and analytics on RFP trends over time.

## Installation
To set up the RFP Aggregator locally, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/mccreamatthewj/RFP-Aggregator.git
   cd RFP-Aggregator
   ```
2. Install the required dependencies:
   ```bash
   npm install
   ```
3. Start the application:
   ```bash
   npm start
   ```

## Usage
- Navigate to the application in your browser to view aggregated RFPs.
- Use the search bar at the top to find specific RFPs.
- Set up notifications in the settings to stay updated on new proposals.
